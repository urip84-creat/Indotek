🗺️ Indotek — Aplikasi Web GIS
Kumpulan aplikasi Web GIS berbasis Leaflet.js yang dikembangkan oleh Indotek.  
Data bersumber dari QGIS dan divisualisasikan secara interaktif melalui browser.
---
📁 Daftar Proyek
1. 🛣️ BIUTR — Bandung Intra Urban Toll Road
Trase Rencana Tol Dalam Kota Bandung | Seksi 1 – 6
🔗 Buka Aplikasi
Layer yang tersedia:
Trase BIUTR Rev.2 (Elevated, At Grade, Transition, Tunnel, Pasupati Bridge, Toll Junction)
Titik Stasiun / Simpul (Pasteur → Cileunyi)
Lokasi: Kota Bandung, Jawa Barat
---
2. ⚙️ Pengolahan & Pemanfaatan Tailing PT. Freeport Indonesia
Kabupaten Mimika — Provinsi Papua Tengah
🔗 Buka Aplikasi
Layer yang tersedia:
Kawasan Industri Mimika
Processing Area
Pembagian Mod Ada (55 zona)
Pipa Freeport
Tanggul
Rencana Jalur (Jalan & Conveyor)
Lokasi: Kabupaten Mimika, Papua Tengah
---
✨ Fitur Aplikasi
🌐 Basemap — OpenStreetMap, Satellite, Hybrid (Foto + Nama Jalan), Topografi
🔲 Toggle Layer — on/off per layer
📏 Pengukur Jarak — dalam meter dan kilometer
📍 Koordinat Real-time — Lat/Lng berubah sesuai posisi kursor
🖱️ Info Atribut — klik fitur untuk melihat data atribut
📂 Upload Layer — GeoJSON, SHP (ZIP), KML/KMZ, TAB (ZIP)
---
🛠️ Teknologi
Leaflet.js v1.9.4
leaflet-measure
shpjs
leaflet-omnivore
Esri World Imagery (Basemap Satellite/Hybrid)
